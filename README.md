# 👑 King Dizzy Digital Bot

A powerful WhatsApp bot built with Baileys.

## ✅ Features
- .menu — Full command list
- .ping — Bot speed test
- .play [song] — Download music
- .dp @user — Get profile picture
- .unlock — Unlock view once media
- .enhance — Enhance image quality
- .news — Latest news
- .ai [question] — Ask AI
- .tagall — Tag all group members
- .kick @user — Kick member
- .add number — Add member
- .promote / .demote — Manage admins
- .mute / .unmute — Control group chat

## 🚀 Deploy on Render.com

1. Upload this folder to GitHub
2. Go to render.com
3. Click "New Web Service"
4. Connect your GitHub repo
5. Set:
   - Build Command: `npm install`
   - Start Command: `npm start`
6. Click Deploy
7. Scan the QR code shown in logs
